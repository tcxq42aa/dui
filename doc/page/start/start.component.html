简要介绍 LUI，以及如何下载、使用，等等。

下载

LUI （当前版本 v1.0.0）提供以下几种方式帮你快速上手，每一种方式针对具有不同技能等级的开发者和不同的使用场景。继续阅读下面的内容，看看哪种方式适合你的需求吧。

生产环境

编译并压缩后的 CSS、JavaScript 和字体文件。不包含文档和源码文件。

源码

Sass、Typescript 和 字体文件的源码，并且带有文档。需要 Sass、Typescript 编译器和一些设置工作。

使用外部脚本

            
    <script src="http://dui.dooioo.com/public/lui/lib/angular.js"></script>

通过 npm 进行安装

            
    $ npm install lui
